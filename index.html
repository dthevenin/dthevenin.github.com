Started my carrier as researcher in Humman Computer Interaction. My main interests are in UI adaptability and mobile computing.

I use the cross-platform technologies promoted by the W3C to develop highly interective applications, that can run on any kind of devices.
Then I implemented or experimented solutions such as VS-Toolkit [2009-2014], GL Engine [2014-2015] or the Animated transitions [2015].


In an other hand I'm exploring how we can make easier the collaboration between the designer, the developer and the tester.
For that purpose I've developed the software ViniSketch Designer - 2009-2015. I also developed technologies for live design and coding to reduce the gap between coding and testing into a real device.
I've also investigated the DAG formalism in order to express data dependencies into a software. As result I've implemented into the VSToolkit an one way dataflow, compiler and optimizer.
This toolkit has been a big optunity to implements concepts such as:

  • Mainloop, events ordering, messages
  • Task scheduling (PAR and SEQ)
  • Dataflow with optimisation, ordering and cycle detection
  • OOP with JavaScript
